Project Name: 
Cart

Synopsis:
Simple example of cart in e-commerce. Program contains page with list of products which can be added to cart, and page with cart itself where user can remove added items.

Application requirement: 
Vuejs:
    "core-js": "^3.6.5",
    "vue": "^3.0.0",
    "vue-router": "^4.0.14"


For Developers
-----------------
    1. Run "npm install" to install VueJS dependencies
    2. Run "npm run serve" for development server or "npm run build" for production

For End-Users
    1. Add desired items to cart
    2. Click on cart
    3. Remove unwanted items from cart
